# Introduction:
Cancer research involves analyzing complex gene expression data to understand how the immune system behaves in different tumor environments. However, such analysis usually requires advanced technical knowledge, making it difficult for many users to perform and interpret results.
This project aims to develop a web-based system that uses artificial intelligence to predict immune states (such as inflamed or immune-excluded) from gene expression data. The system will highlight important genes, provide visual insights, and generate explainable predictions using machine learning techniques.

In addition, the proposed system will integrate a Large Language Model (LLM)-based reasoning module to automatically interpret biological findings and generate human-readable explanations from prediction results, immune scores, and important gene patterns. The LLM module will assist users by summarizing complex biological insights, explaining immune behavior, and providing AI-assisted reasoning for multi-cancer analysis.
The proposed system is an extension of prior work in immune state modeling, enhanced into a user-friendly and explainable AI platform for practical biomedical analysis and research support.
